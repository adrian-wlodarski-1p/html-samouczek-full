Listy

Listy pozwalają wylistować elementy. Oto lista nieuporządkowana:

<ul>
  <li>Element 1</li>
  <li>Element 2</li>
  <li>Element 3</li>
</ul>
  • Element 1
  • Element 2
  • Element 3

A oto lista uporządkowana:

<ol>
  <li>Element 1</li>
  <li>Element 2</li>
  <li>Element 3</li>
</ol>
  1. Element 1
  2. Element 2
  3. Element 3

Istnieją też tzw. listy definicji:

<dl>
  <dt>Herbata</dt>
  <dd>Jest to taki napój.</dd>
  <dt>Kawa</dt>
  <dd>Jest to taki inny napój, ale z kofeiną.</dd>
</dl>
Herbata
Jest to taki napój.
Kawa
Jest to taki inny napój, ale z kofeiną.
