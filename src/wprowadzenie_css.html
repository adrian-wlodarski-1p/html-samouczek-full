Wprowadzenie do CSS

W poprzedniej sekcji nauczyliśmy się podstaw HTML. Były one wystarczające do tworzenia prostych stron, lecz - bądźmy szczerzy - nie wyglądały one rewelacyjnie. Dlatego w tej sekcji nauczymy się CSS, języka stylów.

Co to CSS?

CSS to specjalny język służący do definiowania stylów. Style to sposób, w który napisana w HTML strona się wyświetla. Dokument CSS definiujący style nazywamu arkuszem stylów. Każda strona ma style, nawet jeśli są one po prostu domyślnymi stylami przeglądarki. Te ostatnie nie są jednak zalecane, bo strona może wtedy wyglądać inaczej na każdej przeglądarce (a poza tym domyślny wygląd nie jest zbyt ładny).

Podstawy

Arkusz stylów składa się z tzw. reguł. Każda reguła składa się z selektora i par właściwość-wartość, czyli deklaracji:

selektor { właściwość: wartość; }

Część w klamrach to definicja. Przeanalizujmy więc przykładową regułę:

p { color: blue; }

Jej selektor to p, co oznacza, że ta reguła będzie zaaplikowana wszystkim znacznikom <p>, lecz tylko im. W definicji mamy tylko jedną deklarację. color: blue oznacza, że wszystkie elementy objęte danym selektorym mają niebieski kolor tekstu. Więc, ta reguła zmienia kolor wszystkich znaczników <p> na niebieski.

Kaskadowość

Jest kilka sposobów na dodanie (arkusza) stylów do dokumentu, i są one ułożone w ścisłej hierarchii ważności określającej, które style się bardziej liczą, zwanej hierarchią stylów (zaczynając od rozważanego najpierw, czyli najmniej ważnego):
  1. Domyślne style przeglądarki.
  2. Import arkusza stylów do innego jest skomplikowaną rzeczą, którą się zajmiemy później.
  3. Zewnętrzny arkusz stylów importuje się za pomocą: 
    <link rel="stylesheet" href="...">

    Jest to najszerzej stosowana metoda dodawania arkusza stylów w prawdziwych stronach internetowych (z tego, co wiem).

  4. By użyć wewnętrznego arkusza stylów, trzeba to dodać do <head>: 
    <style>
...
</style>

    Jest to jedna z najwygodniejszych i najbardziej popularnych metod dodania arkusza stylów.

  5. Styl lokalny - styl dodawany bezpośrednio do znacznika za pomocą atrybutu style: 
    <span style="color:blue;">Hello!</span>

Zanim zaczniesz

Zasada, która jest bardzo ważna, ale o której (jak słyszałem) zawsze zapominają nowicjusze, jest to, że PO KAŻDEJ DEKLARACJI MUSI BYĆ ŚREDNIK. Oto przykład złego kodu, który natychmiast wywali wygląd strony do góry nogami:

p {
  color: red
  background: blue
}

Oto właściwa wersja tamtego skandalicznego kodu:

p {
  color: red;
  background: blue;
}

Choć dozwolone jest nie umieszczanie średnika po ostatniej deklaracji, lepiej tego nie robić, tak na wszelki wypadek.

By już nie popełniać takich błędów, możesz użyć Walidatora CSS.

I ostatnia rzecz: w CSS też istnieją komentarze. Będą one ignorowane przez CSS.
